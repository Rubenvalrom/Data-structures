# FunciÃ³n para trazar el camino desde un nodo dado hasta el nodo de inicio utilizando el array de predecesores
def trazar_camino(predecesores, nodo_inicio, nodo_fin):
    camino = []
    nodo_actual = nodo_fin
    while nodo_actual != nodo_inicio:
        camino.append(nodo_actual)
        nodo_actual = predecesores[nodo_actual]
    camino.append(nodo_inicio)
    camino.reverse()
    return camino

Dijkstra

CaracterÃ­sticas:
  1. Aplicabilidad: Funciona en grafos dirigidos y no dirigidos.
  2. Pesos Positivos: Solo funciona con grafos que tienen pesos no negativos en las aristas.
  3. Complejidad Temporal: \(O(V^2)\), donde \(V\) es el nÃºmero de vÃ©rtices en el grafo.
Pasos:

  1. InicializaciÃ³n:

    • Establece la distancia al nodo origen como 0 y a todos los demÃ¡s nodos como infinito.
    • Marca todos los nodos como no visitados. Establece el nodo origen como el nodo actual.

  2. SelecciÃ³n del Nodo Actual:

    • Selecciona el nodo no visitado con la distancia mÃ¡s pequeÃ±a como el nodo actual.

    Fuente: Datacamp

  3. ActualizaciÃ³n de Distancias:

    • Para el nodo actual, considera todos sus vecinos no visitados y calcula sus distancias tentativas.
    • Compara la distancia reciÃ©n calculada con la distancia actual asignada y asigna el menor valor.

    Fuente: Datacamp

  4. Marcar Nodo como Visitado:

    • Una vez considerados todos los vecinos del nodo actual, marca el nodo actual como visitado. Un nodo visitado no serÃ¡ revisado nuevamente.

    Fuente: Datacamp

  5. RepeticiÃ³n:

    • Si el nodo destino ha sido marcado como visitado o si la distancia mÃ¡s pequeÃ±a entre los nodos no visitados es infinita, el algoritmo termina.
    • De lo contrario, selecciona el nodo no visitado con la distancia mÃ¡s pequeÃ±a, establece este nodo como el nuevo nodo actual y repite el proceso.

def dijkstra_algorithm(graph, start_node):
    unvisited_nodes = list(graph.nodes)  # Cambio de metodo de nodos

    # We'll use this dict to save the cost of visiting each node and update it as we move along the graph   
    shortest_path = {}
 
    # We'll use this dict to save the shortest known path to a node found so far
    previous_nodes = {}
 
    # We'll use max_value to initialize the "infinity" value of the unvisited nodes   
    max_value = sys.maxsize
    for node in unvisited_nodes:
        shortest_path[node] = max_value
    # However, we initialize the starting node's value with 0   
    shortest_path[start_node] = 0
    
    # The algorithm executes until we visit all nodes
    while unvisited_nodes:
        # The code block below finds the node with the lowest score
        current_min_node = None
        for node in unvisited_nodes: # Iterate over the nodes
            if current_min_node == None:
                current_min_node = node
            elif shortest_path[node] < shortest_path[current_min_node]:
                current_min_node = node
                
        # The code block below retrieves the current node's neighbors and updates their distances
        neighbors = graph.neighbors(current_min_node)
        for neighbor in neighbors:
            tentative_value = shortest_path[current_min_node] + graph[current_min_node][neighbor].get('weight', 1)
            if tentative_value < shortest_path[neighbor]:
                shortest_path[neighbor] = tentative_value
                # We also update the best path to the current node
                previous_nodes[neighbor] = current_min_node
 
        # After visiting its neighbors, we mark the node as "visited"
        unvisited_nodes.remove(current_min_node)
    
    return previous_nodes, shortest_path
